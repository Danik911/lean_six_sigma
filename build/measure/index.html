Measure Phase

Phase Overview

The Measure phase focuses on gathering baseline data about the current process performance. During this phase, we validated our measurement system, collected data on inventory discrepancies, and established a performance baseline to measure future improvements against.

Measurement System Analysis

Our measurement system analysis (MSA) ensured that our data collection methods were reliable and accurate. The attribute agreement analysis assessed:

  • Agreement between appraisers when categorizing inventory items
  • Consistency of individual appraisers across multiple trials
  • Accuracy of appraisers compared to known standard values

MSA Methodology

We conducted our MSA using the following approach:

  1. Selected 30 representative inventory items with known status (accurate/discrepant)
  2. Had 5 inventory specialists evaluate each item twice, without knowledge of previous results
  3. Analyzed the results using Minitab's Attribute Agreement Analysis
  4. Set acceptance criteria at 90% agreement for all metrics

Measure Phase Outcomes

Our measurement system analysis confirmed that our data collection methods are sufficiently reliable, with good inter-rater agreement. The baseline data revealed that inventory discrepancies affect approximately 18% of items, with certain product categories and locations showing significantly higher rates of discrepancy.

Key metrics established during this phase include:

  • Overall inventory accuracy rate: 82%
  • Average size of discrepancy: ±12 units
  • Financial impact of discrepancies: $125,000 annually
  • Process sigma level: 2.4

These findings provided a clear quantitative understanding of the problem and established the foundation for our analysis phase.
